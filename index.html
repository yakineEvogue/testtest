Commandes de base

Initialise un projet :
git init
Voir le status des fichiers :
git status
Ajouter un fichier :
git add 'fichier'
Ajouter tous les fichiers :
git add .
Commit les fichiers ajouté :
git commit
un commit necessite forcémentun message
Commit avec le message :
git commit -m "message"
Commit + add + message (quand le fichier a déjà été ajouté une fois) :
git commit -a -m "message"

Correction de fichier

Corriger les modification :
git checkout 'fichier'
Retirer le fichier du add :
git reset 'fichier'
Supprimer le dernier commit
git reset HEAD^
Supprimer les deux derniers commit
git reset HEAD^^

La notion de Branches

Voir les branches :
git branch
Créer une branche :
git branch 'nom'
Selection une branche :
git checkout 'nom'
Merger une branche (dans master) :
git merge 'nom'
Conflit de branche
Différence entre deux branches :
git diff nom..master
Mettre le contenu d'une branche dans master :
git fetch . nom
